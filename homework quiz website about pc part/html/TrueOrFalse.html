សំណួរត្រូវ ឬ ខុស

១. CPU គឺជាខួរក្បាលនៃកុំព្យូទ័រ។?

២. RAM ត្រូវបានប្រើសម្រាប់ផ្ទុកទិន្នន័យអចិន្ត្រៃយ៍។

៣. HDD មានល្បឿនអាន/សរសេរលឿនជាង SSD

៤. GPU មានតួនាទីក្នុងការកែច្នៃក្រាហ្វិក និងរូបភាព។

៥. PSU មានភារកិច្ចផ្គត់ផ្គង់ថាមពលឲ្យផ្នែកទាំងអស់ក្នុងកុំព្យូទ័រ។

៦. Motherboard គឺជាក្ដារដែលភ្ជាប់ផ្នែកទាំងអស់នៅក្នុងកុំព្យូទ័រ។

៧. Heatsink និង Fan ត្រូវបានប្រើសម្រាប់បង្កើនកំដៅ CPU។

៨. SSD ប្រើឌីសមូលដើម្បីផ្ទុកទិន្នន័យ។

៩. ស្ពីកឃ័រ (Speaker) គឺជាផ្នែក Input Device។

១០. Wi-Fi Card អាចអោយកុំព្យូទ័រភ្ជាប់ទៅអ៊ីនធឺណិតដោយគ្មានខ្សែ។
